\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}

This report presents the work carried out as part of my final-year engineering
project at CGI Technologies and Solutions Morocco. The project focuses on the
design and development of an intelligent, flexible, and governed platform
dedicated to software migration automation.

In many organizations, application modernization is still largely performed
manually. Developers must analyze legacy applications, identify compatibility
issues, update dependencies, modify source code, resolve compilation errors,
fix test failures, and perform repeated debugging cycles before obtaining a
stable migrated application. This process consumes significant engineering
resources, increases delivery time and costs, and introduces additional risks
of human error.

The proposed project addresses this challenge through the development of a
Migration Factory capable of orchestrating the different stages of software
modernization. The solution combines deterministic transformation mechanisms,
large language models, multi-stage orchestration, failure detection and
evidence collection, independent AI-based repair review, and human governance.

The global vision is to build an extensible migration system capable of
supporting different programming languages and frameworks. Java and Spring Boot
represent the first concrete use case, while other ecosystems such as Angular
are considered as future extensions.

\textbf{Keywords:} Software Migration, Application Modernization,
Generative AI, Large Language Models, Migration Factory, Java,
Spring Boot, Automation, Human-in-the-loop, Software Repair.
